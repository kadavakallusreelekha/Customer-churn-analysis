# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer churn to identify key factors influencing customer attrition and to support data-driven retention strategies.

## Objective
- Identify churn drivers
- Analyze customer behavior and usage patterns
- Segment customers based on churn risk
- Improve customer retention strategies

## Tools & Technologies
- Python (EDA) 
- Power BI
- Data Cleaning & Analysis

## Data Analysis Process
- Cleaned and validated customer demographic, service usage, and billing data
- Performed exploratory data analysis (EDA)


## Power BI Dashboard
The dashboard visualizes:
- Churn rate and trends
- High-risk customer segments
- Service usage impact on churn
- Retention and billing KPIs

## Key Insights
- Higher churn observed among short-tenure customers
- Customers with higher monthly charges showed increased churn risk
- Certain service usage patterns strongly correlated with churn

## Recommendations
- Improve onboarding experience for new customers
- Target high-risk segments with retention offers
- Optimize pricing and service plans

## Conclusion
This analysis provides actionable insights to reduce churn and improve long-term customer retention through data-driven decision-making.
